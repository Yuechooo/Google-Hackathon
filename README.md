# CinemaMatch NYC

An AI-powered platform to help independent filmmakers find their audience in NYC.

## Setup

1. Create a virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Set up your `.env` file with your `GEMINI_API_KEY` (see the `.env` file in the project).
4. Run the app:
   ```bash
   python app.py
   ```
